# off-policy-for-toy-llms

Code & notes on the Off-Policy for Toy LLMs project

## Project description

**Topic**: Off-Policy for Toy LLMs

**Description**: This project explores off-policy RL methods and pitfalls for small LLMs in computationally restricted setting assuming only one mid-tier GPU available for this study (RTX 3090 or RTX 4090). The goal is twofold. First, we need to implement our own research-friendly (i.e., hackable and observable) asychronous reinforcement learning framework. Second, we need to study the tradeoff between on-policiness and off-policiness in asynchronous RL. We will do it using only small (or even tiny) LLMs (from 0.1B to 1B) and very restricted RL tasks (e.g., simple sentiment analysis). We will also need to design crystal clear controllable experiments which would measure the effect of off-policiness on RL training stability, the final accuracy, computational expenses etc.
